How This Works

Nginx Web Server

This site is served by Nginx - a fast, efficient web server that delivers static files.

Browser-Based

All file operations happen in your browser. Files are stored in memory temporarily.

No Backend

No server-side processing. Everything runs locally in your web browser.

Download to Save

To keep files permanently, download them to your computer.

Nginx Server Info

Server: nginx/1.18.0
Port: 8080
Document Root: /var/www/html
Static File Serving: Enabled

Note: This is a demonstration. Files are stored in browser memory only. For production use, you would need a backend server.

Help & Instructions

Getting Started

  1. Click "Select Files" or drag files to the upload area
  2. Click "Upload Selected Files" to add them to the list
  3. Use the search box to find files by name
  4. Click the download button to save files to your computer

Important Notes

  • Files are stored in your browser's memory only
  • Close the browser tab to clear all files
  • Download files to save them permanently
  • This is served by Nginx as static content

Nginx Configuration

This site is configured to be served from:

/var/www/html

Nginx handles all file serving and static content delivery.
